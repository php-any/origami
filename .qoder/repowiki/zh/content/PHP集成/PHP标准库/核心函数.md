### 变量处理函数

#### isset
- 功能：检测变量是否已设置且不为null
- 参数：单个变量（支持AST值）
- 返回：布尔
- 异常处理：计算AST值时若发生错误（如未定义变量），返回false而不抛异常
- 兼容性：与PHP行为一致，抑制未定义变量错误
- 性能：短路判断，避免不必要的类型转换

```mermaid
flowchart TD
Start(["进入 Call"]) --> GetParam["读取第一个参数"]
GetParam --> IsAST{"参数是 ASTValue ?"}
IsAST --> |是| EvalAST["计算 AST 值"]
EvalAST --> ASTErr{"计算是否出错？"}
ASTErr --> |是| RetFalse["返回 false"]
ASTErr --> |否| CheckNil{"值是否为 nil？"}
CheckNil --> |是| RetFalse
CheckNil --> |否| IsNull{"是否为 NullValue？"}
IsNull --> |是| RetFalse
IsNull --> |否| RetTrue["返回 true"]
IsAST --> |否| NilCheck{"参数是否为 nil？"}
NilCheck --> |是| RetFalse
NilCheck --> |否| TypeCheck{"类型是否为 NullValue？"}
TypeCheck --> |是| RetFalse
TypeCheck --> |否| RetTrue
```

### 字符串/路径处理函数

#### basename
- 功能：返回路径中的文件名部分，可选移除指定后缀
- 参数：path（必填），suffix（可选，默认空字符串）
- 返回：字符串
- 异常处理：不抛异常；空路径返回空字符串
- 兼容性：支持Windows风格分隔符，自动去除末尾分隔符
- 性能：字符串处理开销低，O(n)取决于路径长度

```mermaid
flowchart TD

PathEmpty --> |是| RetEmpty["返回 ''"]
PathEmpty --> |否| TrimTrailing["去除末尾 /\\ 并判断是否为空"]
TrimTrailing --> HasSep{"是否包含分隔符？"}
HasSep --> |否| NoSep["返回完整路径"]
HasSep --> |是| Extract["截取最后一个分隔符之后的部分"]
Extract --> SuffixChk{"suffix 是否非空且位于末尾？"}
SuffixChk --> |是| TrimSuffix["移除后缀"]
SuffixChk --> |否| Keep["保留原名"]
TrimSuffix --> Done["返回结果"]
Keep --> Done
```

#### realpath
- 功能：返回规范化绝对路径；若路径不存在或不可解析，返回false
